Tailscale Auto-Join QR

วิธีใช้งาน:
1. อัปโหลดไฟล์ทั้งหมด (index.html, qrcode.min.js) ไปยังโฮสต์ เช่น Netlify หรือ GitHub Pages
2. เปิดหน้าเว็บ index.html ด้วยมือถือ
3. กด Install เพื่อติดตั้ง Tailscale App (iOS / Android)
4. กด Join Now หรือสแกน QR Code → แอปจะ Auto-Join ด้วย Auth Key ที่ฝังไว้

หมายเหตุ: Auth Key มีอายุจำกัด (เช็คจาก Tailscale Admin)
